# All the apps

Standard offering server-worker demonstrating use of cloud native:

- Authentication
- SQL Database
- Blob storage
- Cache
- Queue
- Schedule
- Signed URLs
- Static assets
- Monitoring
- Errors

These come in very specific combinations and target the serverless platform of each cloud provider.

## Combinations

- gcp-django
- aws-django
- gcp-rails
- aws-rails
